# 🚀 Digital Premium tool

## 📌 Project Name

**Digital Premium tool**

## 📝 Description

Premium Digital Tools is a modern web application that provides users with easy access to digital subscriptions for daily and lifetime use. The platform is designed to be fast, responsive, and user-friendly, helping users manage and explore digital tools effortlessly.

## ⚙️ Technologies Used

- Vite + React.js
- Tailwind CSS
- React Toastify
- Lucide React
- React Icons

## ✨ Features

- 🛒 Add to Cart functionality
- ❌ Delete/Remove items from cart
- 📱 Responsive Menu Toggle (Open/Close Navigation)

## 🎯 Purpose

The main goal of this project is to simplify access to premium digital subscriptions while ensuring a clean UI and smooth user experience.

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/sagorsaha00/PremiumTools_explore.git

live link:https://digital-premium-tools.netlify.app/


# Install dependencies
npm install

# Run the project
npm run dev
```
